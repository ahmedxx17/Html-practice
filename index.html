Practicing HTML <head> Tags

In this file, I'm practicing all the important head tags used in HTML.

  • <meta> – for metadata
  • <title> – sets the title of the tab
  • <base> – sets a base URL
  • <link> – links external files like CSS
  • <style> – used for internal CSS
  • <script> – used to run JavaScript
